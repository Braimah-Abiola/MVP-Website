Pygame

Pygame is a multimedia library for Python for making games and multimedia applications. It is a wrapper around the SDL(SimpleDirectMediaLayer) library. Before you can install pygame on your PC you need Python already installed. If you don't know how click HERE.

How to Install on Pygame On Windows

After installing Python open the command line or cmd and enter the following code.
To open cmd or commandline on Windows or PC :

    Press Windows Button + R, Type cmd into the run window and press Enter OR
    Press Windows Button + R, Type cmd into the run window and press Enter OR
    Press Windows Button + R, Type cmd into the run window and press Enter OR


					pip install pygame

How to Install on Pygame On Mac & Linux

On MacOS and Linux open the terminal and enter the following code.
To open the terminal on MacOS and Linux :

    Press Windows Button + R, Type cmd into the run window and press Enter OR
    Press Windows Button + R, Type cmd into the run window and press Enter OR
    Press Windows Button + R, Type cmd into the run window and press Enter OR


					pip3 install pygame

Verifying Sucessful Installation

After successfully installing Pygame you will see the below message in the terminal or commandline 

					pip install pygame

How to Use Pygame

Open your favourite code editor or Python IDE and enter the following Python code to start using Pygame 

					import pygame

After inputing the line of code run it in the code editor or Python IDE. If Pygame was installed properly you should see the message below in the Commandline or Terminal depending on the OS you are using

					pygame 1.9.6  #This indicates the version of pygame installed
					Hello from the pygame community. https://www.pygame.org/contribute.html
